## 🔮 What Is This Sorcery?

**XAI-RainCast** is a hybrid **CNN-LSTM / Transformer** rainfall prediction engine that doesn't just spit out a number and vanish into the mist — it **shows its work**, quantifies **uncertainty**, and hands you the receipts via full **explainability (XAI)**.

Built during a summer research internship at **SRMIST**, under the guidance of **Dr. P. BalaMurugan**, this isn't your average weather widget. This is a meteorological black box that got tired of being a black box.

```
     ☁️  →  🧠  →  📊  →  ☔
   Clouds   Model   Explains   Rain
            (finally)  itself   (or not)
```

## 🚀 Features That Hit Different

| ⚡ Feature | 🌪️ What It Does |

| 🧬 **Hybrid CNN-LSTM/Transformer** | Spatial + temporal patterns, fused into one prophecy engine |
